The Project
If you haven't read about this class on my blog, long story short I was Mr.Harris's top student. I excelled 1000% percent in his class compared to other students, and so we discussed me taking one of his old 2D floorplan project's I did the previous semesters, creating 3D models for everything, applying textures and creating HD-renders. Mr. Harris let me go at it.
How I started
So first we choose which floorplan to do. I really don't remember how we choose one, but we did choose one. Now keep in mind, I did this all in AutoCAD. AutoDesk 3DS Max is really more suitable for 3D models, but AutoCAD can do it if you know how to do it. Things such as walls are easy, I really just have to use the pull tool and pull them to the appropriate height. But other models require much more work.
I started with the Kitchen, since I figured it has lot's of things going on. You have a sink, fridge, oven, island, cabinets, etc. All things to model!. After that I went onto the bathroom, and soon other places in the house. Note I came up with some newer designs after HS, so you can see a difference in the different renderings.
