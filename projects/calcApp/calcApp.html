Calculator App for Android

This is an app I made for the Android environment. As the name states, it's a calculator. Below is a video.

Software and Hardware
This app was developed using Android Studio and deployed for testing on the LG V20
Development
This app was developed using Java. I based the calculator off of what I thought as a simple calculator. The four main arithmetic operations, digits 0-9, clear result and equals sign. I realized after I should of included a decimal point. That will be soon to come.
How did I make it?
Generally, the developing of making this is similar to an interactive program for a desktop/laptop. The difference is the way of receiving inputs, like touch of a button from the finger, and the screen layout. The app displays the numbers as strings on the screen, but behind the scenes, it is converted to the appropriate data type, such as integer, double, etc. From there, the result is saved, and is converted to a string type to be displayed on the screen.
