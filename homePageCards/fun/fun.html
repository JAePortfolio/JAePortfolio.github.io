TF2 Sentry
This is a project I'm doing for fun and to learn from. I am building a functioning model of a sentry from the game Team Fortress 2. It will be made from cardboard, PVC pipe, held together with various bolts, cut out slots, etc.
It will be powered by an ARM Cortext-M7. I am using a ST32F7 Nucleo Board to develop on to interface with sensors, motor(s) and other components. It will feature an Android app as well, where the app will represent the in-game PDA which tells me the status of the sentry (health, enabled/disabled, etc). It will communicate with the sentry via Bluetooth Low Energy.
The project is moving a bit slow as it's hard to build this without the proper equipment, but it's moving along daily. More to come to the page soon...
